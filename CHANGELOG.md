# Changelog

## 0.1.0

- Added operating model, planning, checkpoint, frontend boundary, TypeScript,
  debugging, refactor, security, performance, and review common rules.
- Added `frontend-bugfix-debugger`, `frontend-refactor-surgeon`, and
  `frontend-quality-reviewer`.
- Integrated screenshot-to-frontend routing with architecture, lint, visual QA,
  and quality review gates.
- Added Codex and Claude distribution target scripts and validation checks.
- Added plugin metadata, install documentation, and demo workflow examples.
