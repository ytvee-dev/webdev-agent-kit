# Установка

Скачайте архив `webdev-agent-kit-claude-code.tar.gz`, распакуйте его в `.agents/` и создайте root `CLAUDE.md`, который указывает на `.agents/AGENTS.md`.

После установки запустите `адаптируйся`.
