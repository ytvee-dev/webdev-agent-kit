# TypeScript Rules

- Preserve strict mode assumptions.
- Use discriminated unions instead of boolean state bags when behavior branches.
- Keep public types descriptive and stable.
- See `typing-patterns.md` for `satisfies`, narrowing helpers, and boundary
  typing patterns.
