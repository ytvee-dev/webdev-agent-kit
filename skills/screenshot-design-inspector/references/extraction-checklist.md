# Extraction Checklist

- Identify whether each screenshot is desktop, tablet, or mobile.
- Read visible typography: size, family, weight, line height if inferable.
- Read visible color roles: text, background, border, accent.
- Read spacing rhythm: gaps, padding, section spacing, card density.
- Read breakpoint intent: what collapses, stacks, hides, or expands.
- Compare typography and spacing separately for desktop, tablet, and mobile
  screenshots.
- Record confidence for every extracted typography, color, and spacing value.
- Record uncertainty explicitly instead of guessing.
